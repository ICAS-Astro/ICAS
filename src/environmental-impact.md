# Impact Environemental

- Hosting

Top 5 Green Data Centers (à choisir dans un objectif d’éco-responsabilité):

1. Sweden (Suède).
2. Iceland (Islande).
3. Norway (Norvège).
4. Finland (Finlande).
5. Denmark (Danemark).

Sources: https://www.greenmatch.co.uk/blog/top-countries-with-sustainable-data-centres

- Data Transfer

Transfert de données:
  - Images (.raw / .tiff / .jpg / etc) -> réseau local / USB. Pas besoin
    d'échange de données via un serveur.

  - Configuration application (paramètres de profil utilisateur) -> possibilités:
    1. Base de données utilisateur -> infrastructure (Backend & Database)
	2. Exportation de configuration vers un fichier / dossier (compressé) ->
      Pas besoin d'infrastructure.

- Compute
Pas d’utilisation de Lambda Serverless pour l’hébergement d’un site web vitrine.

**Measuring Impact**:

**EcoIndex**: Estimate the performance and footprint of your web pages.
**Carbon Footprint**: Estimate the CO2eq emissions of your infrastructure using
calculators (e.g., Cloud Carbon Footprint).

- https://www.ecoindex.fr

**Target**: Provide an Eco-score estimation or a list of specific optimization
choices made to reduce energy consumption.
